1. Functions

Javascript is a popular programming language and it is essential to use Js especially when you are developing anything meaningful on the web or a mobile. In Js, the use of functions is inevitable and a programmer should get used to it. Functions in javascript helps us in structuring large programs and reduces the repetitions, thus making our programs to look smarter and readable. However, most programmers finds it hard when working on function() in Js. Here are the tips that will help you out:


Functions

Declaring Functions

In Js , always start the declaration of the functions with a keyword function()

  • Functions names in Js can have combinations of digits, letters, underscores and dollar
  • The codes in a function must be enclosed in a block of curly braces { }
  • List of names that goes inside the function in Js is called function parameters
  • During the invoking of the functions, the real values that are received are called function arguments

Invocation of Function()

Javascript functions will start performing the task intended only after being invoked. This in turn will reciprocate the execution of the functions. A function in Js can be invoked by the following events:

  1. When it is invoked by the user for instance by clicking a button
  2. When it is invoked by the browser for instance ; after page finishes loading.
  3. Other javascript functions can self-invoked themselves automatically


Use of Return Statements in Functions

Return statement is a keyword used in javascript and other programming languages to indicate that the program should stop executing after certain conditions have been met

JavaScript functions in most cases reckon a return value which is returned back to where and who was calling
