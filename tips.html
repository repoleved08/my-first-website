Coding Tips for Beginners

Most beginners consider JavaScript to be the hardest programming language, this is sometimes true if one starts the programming journey with javaScript. This is because one finds it hard to adjust the mindset and focusing on the logics that lies behind every language. In most cases, it is advisable for a beginner to get to know the basics of Hypertext Markup Language (HTML) and Cascading Style Sheet (CSS) before diving into javascript. This is because it serves as a mind-openers and introduces the basics of programming

In addition, there are available resources found on the internet for instance: a book called Eloquent Javascript and You Don't Know JavaScript



Coding Tips for Intermediate Programmers

Computer programming works by "Learn by doing" technique.

In the intermediate level, most programmers seems to give up and quit their programming career. this issue come up because according to them, they feel like they are not making any progress at all. In order to avoid this, a programmer should learn to adopt the following techniques:

  • Work with others

    This increases your capability to master well most of the topics that seems to be tedious when working on them alone

  • Interact with others' Codes

    Interacting with other people's codes on open sources platforms such as Github, stackoverflow, Reddit and Quora. This sites contain millions of questions that almost all programmers go through. I f you go through this platforms, you will find answers to the questions that you have been asking yourself.

  • Help the newbies

    This is one of the best way to upgrade you skills. Sharing your skills with noobs helps an intermediate programmer to master the techniques in a faster way

  • Ask The Questions

    Asking questions in programming will build your confidence and helps in the maintenance of your code. A question asked is another step forward. Asking questions should come in after one had done research and no solution has been attained
